FAVICON SETUP GUIDE
===================

Place your favicon images in this folder with these exact names:

Required Files:
--------------
1. favicon.ico (16x16 or 32x32 pixels)
   - Traditional favicon format
   - Supported by all browsers

Optional but Recommended:
------------------------
2. favicon-16x16.png (16x16 pixels)
   - Small favicon for browser tabs

3. favicon-32x32.png (32x32 pixels)  
   - Standard favicon for most displays

4. apple-touch-icon.png (180x180 pixels)
   - For iOS home screen icon
   - Should be square with rounded corners handled by iOS

Image Format Recommendations:
-----------------------------
- Use PNG format for best quality
- Use transparent background if your design allows
- Keep the design simple and recognizable at small sizes
- Use high contrast colors

How to Create:
--------------
1. Design your favicon at 512x512 pixels first
2. Export/resize to the dimensions listed above
3. Save with the exact filenames shown
4. Place all files in this 'public' folder

Online Tools (if needed):
-------------------------
- favicon.io - Generate from text, image, or emoji
- realfavicongenerator.net - Generate all sizes at once
- canva.com - Design custom favicons
